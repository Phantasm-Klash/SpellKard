# base

Reserved for original base theme assets.

See `theme_manifest.json`. Production media must also be listed in `../../assets/asset_manifest.json`.

Current placeholders:

- `ui/lobby_standee.svg`: self-authored original lobby standee placeholder for the base theme. It is tracked in `../../assets/asset_manifest.json`.
- `ui/mode_card_frame.svg`: self-authored original frame art for mode, queue, community, and settings cards.
- `ui/collection_card_frame.svg`: self-authored original frame art for deck, chest, replay, and reward cards.
